# Glossary

## Essentials

Neste vault, "Essentials" significa Pokemon Essentials v21.1 usado como base de referencia e fonte de dados.

## Pack Gen 9

Pacote adicional que amplia a base para cobrir conteudo da nona geracao e exige hotfix especifico para funcionar corretamente.

## Runtime oficial

A build consolidada e jogavel usada para execucao e teste manual.

## Fonte overlay

Camada de arquivos editaveis que sobrepoe a base completa no processo de rebuild.

## PBS

Conjunto de arquivos de dados do ecossistema Essentials usados para descrever especies, moves, items, abilities e outras entidades.

## Engine autoritativa

Motor de batalha em que o servidor, e nao o cliente, determina o estado canonico da partida.

## Event log

Sequencia estruturada de eventos produzida durante a batalha para suporte a replay, debug e resync.

## Snapshot

Fotografia serializada do estado de batalha em um ponto especifico.

## Resync

Processo de realinhar o cliente com o estado autoritativo quando ha atraso, reconexao ou desync.

## Ownership

Clareza sobre qual arquivo, pasta ou camada deve receber determinada alteracao.

## Source of truth

Local ou processo oficialmente reconhecido como referencia correta para uma classe de informacao.
